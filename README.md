# e-plantShopping

Paradise Nursery is a React and Redux Toolkit shopping-cart application for browsing and purchasing houseplants.

## Features

- Paradise Nursery landing page with a Get Started button
- Product catalog organized into three plant categories
- Six unique houseplants per category
- Add to Cart buttons that disable after selection
- Dynamic cart quantity indicator
- Redux-based cart state management
- Increase, decrease, and remove cart items
- Per-item subtotals and overall cart total
- Continue Shopping navigation
- Checkout placeholder message

## Technologies

- React
- Redux Toolkit
- React Redux
- Vite

## Run locally

```bash
npm install
npm run dev
```
